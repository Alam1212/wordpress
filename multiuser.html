Multisites word press allows numerous blogs to exist within one installation. WordPress multi-user makes it possible for websites to host their own blogging communities as well as configure blogs from a single dashboard. Word press multi-sites also adds eight new data tables for each blog.

WordPress is a dynamic tool that can be used to make an assortment of different types of websites. The main use case of word press is to create blogs but it can also be used to create business, e-commerce and educational websites. WordPress blog is a website where multiple users can each have their own blog, and all of the individual blogs will run under a single WordPress installation and on a single domain. The multiuser set-up requires the use of a plug-ins. The first step is to enable multisite on your WordPress installation is by opening the wp-config.php file found on the server(Contact your web hosting provider or webmaster) and add define(‘WP_ALLOW_MULTISITE’, true); After you make the update, click the “Network” link to go to the screen through which you’ll set up your multisite network. You can choose for the addresses of the sites to be structured as subdomains, like: site1 or set up as subdirectories like site1. After you make that choice, you’ll choose a title for your new network and the administrative email address you want to use. Then click the “Install” button and WordPress will help you make a new folder in your main blog folder. When you log back in to your blog (making changes to the wp-config.php and .htaccess files will likely have the effect of logging you out), you’ll see a new Super Admin menu. Click the “Sites” selection in this new Super Admin menu and you’ll be taken to a form that lets you create a new website that will have its own set of logins and its own administrative dashboard.
