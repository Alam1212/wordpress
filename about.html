History

WordPress started in 2003 with the innovation of Mike and Matt Mullenweg who were looking for the capability of producings a personal publishing system. Today about 33% of the websites in the world are architected and designed using the WordPress product. Using the open source ideology, many contributors from the past and today continue add additional features and capabilities to the WordPress product.

The main objective to WordPress is allowing anyone with limited computer skills to develop a website using the Graphical User Interface (GUI) and utility programs that are included in the WordPress product.

primarily 4 Freedoms:

  • 1st Freedom - run the program
  • 2nd Freedom - open source, allowing collaboration,
  • study how the product works and create new features
  • 3rd Freedom - Redistribute WordPress
  • 4th Freedom - Redistribute Modified Versions
